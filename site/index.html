Install from source

Linkwarden

Save and organise links directly to your Linkwarden instance

Download extension.zip
  1. Unzip extension.zip
  2. Open chrome://extensions (Chrome) or about:debugging#/runtime/this-firefox (Firefox)
  3. Chrome: enable Developer mode → Load unpacked → select the dist/ folder
  4. Firefox: click Load Temporary Add-on… → select dist/manifest.json

Links

Sync and manage bookmarks across browsers with Linkwarden integration

Download for Chrome / Edge / Brave
Download for Firefox
  1. Unzip the archive
  2. Chrome: open chrome://extensions → enable Developer mode → Load unpacked → select the extracted folder
  3. Firefox: open about:debugging#/runtime/this-firefox → Load Temporary Add-on… → select manifest.json from the extracted folder
